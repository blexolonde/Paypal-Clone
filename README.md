# PayPal Website Clone

## Overview
This is a front-end clone of the PayPal website, built from scratch as a learning project to practice HTML, CSS, and JavaScript skills.

## Project Details
- **Type:** Front-end only (no backend functionality)
- **Purpose:** Educational/Portfolio project
- **Features:** Responsive design, multi-page navigation, interactive dropdown menus

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Baloo Da 2)

## Pages Included
1. **Home/Landing Page** - Hero banner, services section, features showcase
2. **Login Page** - User authentication interface
3. **Sign Up Page** - Account registration with Personal/Business options

## Key Features
- Fully responsive design (mobile, tablet, desktop)
- Animated hamburger menu for mobile navigation
- Dropdown menus for navigation
- Smooth page transitions
- Clean, modern UI matching PayPal's design language

## Responsive Breakpoints
- Desktop: 1500px+
- Laptop: 1000px - 1499px
- Tablet: 550px - 999px
- Mobile: 400px - 549px
- Small Mobile: <400px

## Installation & Usage
1. Clone  the repository
2. Open `index.html` in your browser
3. No build process or dependencies required

## Limitations
- No actual payment processing
- No backend/database integration
- Forms do not submit data
- Authentication is UI-only
